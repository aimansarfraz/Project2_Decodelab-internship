# E-Commerce Order Dataset - Exploratory Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a cleaned e-commerce order dataset to identify sales trends, customer behavior, product performance, and revenue patterns.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Analysis Performed
- Descriptive Statistics
- Product Performance Analysis
- Revenue Analysis
- Payment Method Analysis
- Referral Source Analysis
- Monthly Sales Trends
- Correlation Analysis
- Outlier Detection

## Key Findings
- Identified top-selling products.
- Analyzed revenue contribution by product.
- Examined customer payment preferences.
- Detected high-value order outliers.
- Evaluated monthly sales trends.
